The Minimum Spanning Tree Problem

Historical Motivation

The Minimum Spanning Tree problem is a fundamental problem in graph theory and computer science that involves finding the minimum weight tree that spans a given graph. The problem can be traced back to the early 20th century and has been studied extensively since then. Czech scientist Otakar Bor≈Øvka developed the first known algorithm for finding a minimum spanning tree, in 1926. He wanted to solve the problem of finding an efficient coverage of Moravia with electricity.

Connection to the project

The goal of the project is to finds a subset of the edges that forms a tree that includes every vertex, where the total weight of all the edges in the tree is minimized. We will use Kruskal and Prim's algorithm algorithm to visualize this. The weights of the edges are based on the Euclidean distance between the two nodes
